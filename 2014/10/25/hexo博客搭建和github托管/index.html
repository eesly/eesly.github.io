关于Hexo及基本安装

1.Hexo是一个基于Node.js的静态博客程序，可以方便的生成静态网页托管在github上。

2.安装Git，安装Node.js。

3.在安装目录上新建一个Hexo\blog的文件夹，在里面右键Git-bash—安装Hexo。
